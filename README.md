
# WebOasis

FOSS Starting page / Search Engine for the web.  
**Originally made by Webby.**
**Forked by 2soup7.**
## Run Locally

Clone the project

```bash
  https://github.com/ParrotDevelopers/WebOasis.git
```

Go to the project directory

```bash
  cd WebOasis
```

Building project

```bash
  npm run build
```

Install dependencies

```bash
  npm install
```

Setting Port
```bash
  export PORT=8000
```

Start the server

```bash
  npm run serve
```
